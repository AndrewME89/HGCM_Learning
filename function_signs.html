Updating Function Signage

To ensure accurate information is displayed on the conference lobby signage, follow this procedure.

Procedure
Lobby Signage:
  1. Open the following website link: https://app.optisigns.com/signIn

  2. Log in using the provided credentials:

    • Username: vincentkhoo@excel.com.sg

    • Password: ess530108

  3. Once logged in, click on the "Files/Assets" link located near the top left corner of the page.

  4. Locate and click on the "Conference Lobby" icon.

  5. The file containing the conference rooms list will open for editing.

    1. Click on the text box beneath the corresponding function room and enter the required Function name.

    2. Complete the necessary fields for the day.

  6. Click the "Save" button located in the top right corner of the page.

  7. After saving, click the "Close" button to return to the "Files/Assets" page.

  8. Repeat the same steps for the "Conference Upstairs" signage.

  9. Once all updates are complete, check the electronic signage in the conference lobby to confirm the details have been updated correctly.

Room Signage:
  1. To log in to the signage, use the following link: https://www.digitalsignage.com/

  2. Use the login details below:

    • Username: excelsg05@gmail.com

    • Password: hgcm6789

  3. From the Homepage, select "Scenes" from the menu on the left.

  4. Select the room number you wish to change.

  5. Open the Layers menu on the left of the screen and select the Function option.

  6. On the right side of the screen, click Label and type in the function name under "displayed text".

  7. Click the save icon along the top of the screen, then the exit button.

  8. Select Players on the left side of screen, then select the required screen to update.

  9. Click the circular icon with the tick in the center to push the new content to the screens.
