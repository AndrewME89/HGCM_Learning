Setting Tasks

Tasks can be set up for guest requests, jobs, or undertakings to be actioned by an individual or department.

Example tasks:
  • Roll Res (B2B)

  • Set up bottle of sparkling prior to arrival

  • Follow up missing chargeback

  • Please place a foam pillow and bathrobes on arrival

  • Please remove foam pillow after guest departure

  • Following up payment

Procedure
Setting up a Task:
  1. Right-click on the folio that requires a task.

  2. Click ‘Add Message’.

  3. Set the type from Message to Task (using the drop-down menu) and ensure Status is “Open”.

  4. Down further is the Action box:

    1. Choose the relevant department or team member to assign the task to.

    2. Choose the correct date the task needs to be actioned on.

    3. In the text box at the bottom, write down the task required.

  5. Click the ‘Next’ button.

  6. The set task will appear in the (1) Corporate tab in the Reservation Message section of the window.

  7. You will see the reservation now has a yellow highlighted box under the T/M column in folio view.

Note:

Tasks can also be set up from the Communication tab.

  1. With the folio open, select the Reservation Message dropdown.

  2. Select Add New.

  3. Follow the above steps to set up a task.

Viewing Tasks
  1. Right-click on the folio that has a Light Globe Icon next to the name.

  2. Click ‘View Message’.

  3. The window below will pop up.

  4. You can check who created the task and when by clicking on History.

  5. Browse through different Tasks under the same folio using the arrows.

  6. Click ‘Cancel’ to close the window.

Note:

Tasks can also be viewed in the message box in the Communication tab.

Closing a Task

Once the task is done/actioned, the person responsible for it must close the task.

  1. Right-click on the folio that has a Light Globe Icon next to the name.

  2. Click ‘View Message’.

  3. The window below will pop up.

  4. Change the status from "Open" to "Closed".

  5. Click ‘Next’ to confirm.

Printing Task List
  1. To print the task list, click Reports → Reservations → Task List by Action Date.

  2. Select today's (or the required) date in both fields.

  3. Select "Open" as the Active Status.

  4. Click OK.

  5. This will generate the task list.
