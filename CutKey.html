Room Keys

Ambiance Access Management Software is the access control system for the hotel's electronic door locks. It allows you to register new guests, create keys, issue additional or replacement keys, modify guest information, and process check-outs. You can also use it to read key card information.

Ambiance also includes the M-unit, used for auditing or programming electronic door locks.

Once logged into Ambiance, you'll see the interface below (which may vary based on user permissions).

Procedure
Guest Registration - Making Room Keys

Reminder: Â Keys should only be cut after the guest has been checked in within iHMS. This prevents issuing keys for rooms that are vacant, dirty, or already occupied.

  1. Click on the Guest Registration icon.

  2. The interface below will appear; click on "New Registration."

  3. Update the check-out date based on iHMS and click "Next to Rooms."

  4. Find the allocated room using the search box and click "Next to Access."

Note:

Multiple rooms can be added to a single key card by selecting and adding them during this step.

  1. By default, guests have access to Level 17, the pool, and the gym. Click "Next to Guests" to proceed.

  2. Enter the guest's first and last name. Verify all information in the summary box. Click "Make Keys."

  3. A pop-up will appear. Click "Start" to create keys. By default, two keys are issued per room.

  4. Place a key card on the reader (with a green light) until you hear a beep. Once finished, click "Done."

  5. Write the room number and departure date on a key wallet and insert the key cards.

Making an Additional Key

If a guest leaves a key in their room or requests an extra one, you can issue an additional key. These are copies and do not invalidate original keys. You can specify the number of keys needed.

Confirm the guest's name, room number, and departure date before issuing additional keys. You may request photo ID for verification.

  1. Click on the Guest Registration icon.

  2. In the interface, click the magnifying glass icon and search for the room number.

  3. Select the correct room and guest. Click on "Additional Keys."

  4. Click "Start" and cut the new key(s). Adjust the number of keys if needed. Once done, click "Done."

Making a Replacement Key

If a guest loses their room key(s), you can issue replacements. This generates new keys for all guests in the registration, invalidating previous keys (physical or mobile) at that access point.

Confirm the guest's name, room number, and departure date before issuing replacement keys. You may request photo ID.

  1. Click on the Guest Registration icon.

  2. In the interface, click the magnifying glass icon and search for the room number.

  3. Select the correct room and guest. Click on "Replacement Keys."

  4. Click "Start" and cut the new key(s). Adjust the number of keys if needed. Once done, click "Done."

Making a Limited Use Key

Limited Use Keys are typically for guests returning after check-out to retrieve belongings. They're valid for a single use or until expiration.

  1. Click on "Limited Use Key."

  2. Choose a room and click on "Make Keys."

Modifying a Guest Registration

Guests might request changes to their check-in/out dates, room, or key access. Each modification requires new keys to be made and issued, invalidating previous ones.

  1. Click on the Guest Registration icon.

  2. In the interface, click the magnifying glass icon and search for the room number.

  3. Select the correct room and guest. Click on "Modify Registration."

  4. Make the necessary changes (e.g., check-out date, room).

  5. Create new keys.

Checking Out Guests from Ambiance

Room keys expire at 12:00 PM on the departure day. Before then, the room is considered occupied in Ambiance, preventing new key creation even if the guest has checked out in iHMS and left.

Check out room keys from the system before 12:00 PM on the departure day.

  1. Click on the Guest Registration icon.

  2. In the interface, click the magnifying glass icon and search for the room number.

  3. Select the correct room and guest. Click on "Check Out."

Reading Keys

The Ambiance key reader, accessible from the main toolbar, displays key status information.

  1. On the main page, click the key reader icon.

  2. In the window that appears, click "Read Key."

  3. Present a key to the encoder. The following details will be displayed:

    • Keycard type

    • Encoded by

    • Encoded on device

    • Key mode

    • Created on

    • Check-in

    • Check out

    • Key ID Status (Active/Expired/Blocked/Canceled)

    • Room access

    • Floor access

    • Encoded for guest

Handling New Guest Arrivals When Room is Unavailable in Ambiance

If a new guest arrives and their room isn't shown as vacant in Ambiance, it might be assigned to another guest.

  1. Click on the Guest Registration icon.

  2. In the interface, click the magnifying glass icon and search for the room number.

  3. Check the guest details assigned to the room.

  4. Cross-check the name in Ambiance against iHMS.

    • If the guest has departed, check out the keys from Ambiance.

    • If another guest occupies the room, inform the Duty Manager and assign a new room to the arriving guest.

Note:

Never assume a room is vacant and check out the old guest from Ambiance without proper investigation!
