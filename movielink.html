Movilink Procedure

Login

Click on the "Internet Explorer" icon to access the "HD-IP System Login" session.

  • ID: hotel

  • Password: staff

Note:
  • Use lowercase letters when logging in.

  • Upon completion, click "Logout" or close "Internet Explorer."

Accounting Report
Billing Report

This report presents all charges generated within a specified period.

  1. Select the desired period using the mini-calendars next to "startDate" and/or "endDate."

Note:
  • Specify a particular room in the "Room No" field or leave it blank to view charges for all rooms.

  • For hotels with an interface, manually post any "UNPOST" charges to the guest folio in the Front Office System.

Billing Report - Movie Detail

To view the duration a guest watched a specific movie, click the "Detail" button next to the corresponding movie posting record.

Minibar Report

This report displays all Mini-Bar charges entered by staff via the television.

  1. Select the desired period using the mini-calendars next to "startDate" and/or "endDate."

Note:
  • Specify a particular room in the "Room No" field or leave it blank to view charges for all rooms.

  • For hotels with an interface, manually post any "UNPOST" charges to the guest folio in the Front Office System.

Room Management

This screen provides critical information regarding the system's operational status:

  1. All rooms configured in the Movielink HD-IP system will be displayed.

  2. The report indicates each room's current status:

    • Check-in

    • Check-out

    • Adult Enable

    • Adult Disable

    • Pay Service Enable (All movies enabled)

    • Pay Service Disable (All movies disabled)

    • A pink heart signifies normal room operation.

    • A grey heart indicates an operational issue. Check power/cable connections, the Set Top Box ID, etc. If the issue persists, in-room equipment might need replacement (contact Hotel Engineering).

  4. To enable/disable Adult/All Movies service:

    • Tick the checkboxes next to the relevant room numbers.

    • Click "Enable Adult," "Disable Adult," "Enable Pay Service," or "Disable Pay Service."

    • The action will be applied to all selected rooms.

Checking In-Room Set Top Box ID
  1. Ensure all connections are secure and turn on the television.

  2. The system's "Main Menu" should appear.

  3. Using the in-room Movielink remote, enter "012340".

  4. If entered correctly, a small input box will appear at the top left corner. Enter "123" and press "OK."

  5. The "engineering screen" will display room details. Verify that the room number matches the physical room.

  6. If correct, use the arrow keys to select "Cancel" and press "OK" to return to the "Main Menu."

  7. If incorrect, select "Reset Room" and press "OK." The Set Top Box will reboot. When prompted, enter the correct room number and press "OK." The system will return to the "Main Menu."
