Creating a Reservation

Private Bookings
Step 1. Get all the important details from guest/booker:
  • Date of stay

  • Room type (bedding configuration)

  • How many pax

  • Check if guest is GC Member and ask for membership number to confirm

Step 2. Check Availability & Rate
  • On HMS

    • Reservations → Rooms Availability → (3) Room Type Tab

    • Adjust date range if needed

    • Check for rates and open Rate Calculator (M: → Operations → Reservations → Promotions → Calculators → Promotional Calculator) to apply discounts.

      This should also be available on every desktop of FO Reception Computer.

    • Ensure to apply additional 10% off to the rates for GC Rewards members.

Step 3. Provide Rate
  • Always offer BAR & SAVER Rates and explain difference between the two rates

  • We can match rates found online only if we are able to find it.

  • Corporate guests can utilise corporate rates for personal stay.

Step 4. Create the booking
  • Press Alt + C or click

  • Fill up necessary information:

    • Guest - locate if guest has an existing profile.

      • Type in guest name in correct sequence (Last Name, First Name) and ensure it is spelled correctly.

      • Press F2 or the elipses icon . New window will pop up with list of guest profile. Choose correct profile by matching contact number/company name. Edit profile and add/update credit card number to guarantee booking.

      • If no existing profile found, create new profile by clicking and ensure all details are filled up:

        • VIP Status

        • Full Name (Last Name, First Name, Title)

        • Contact Number

        • Email (for booking confirmation purposes)

        • Credit card details

        • If GC Member and no existing profile yet, add GC Membership # & ensure VIP status is VIP 4

    • Company Name - locate if guest's company is already in the system

      • Use 'Private Booking'as company if booking directly over the phone or through email.

      • Create company if not yet in the system

    • Room Type - best rate type applicable will be in red coloured font. If multiple rate available, ensure to click on the best rate code applicable to the rate that had been quoted.

    • Check In date

    • Check Out date

    • Number of Adults & Number of Children (only if applicable)

    • Rate type

      • Click on the elipses icon and list of rate code will pop up. Choose the best rate depending on the rate you've quoted to the guest.

    • Contact Person

      • Get the booker's name and contact number and add it under contact person's details

      • If the guest who will be staying is the same person making the booking, type 'self' under contact person's detail

  • Once all details are filled up, press Next twice

    • Mkt Seg 4 & Source will automatically be filled up

Step 5: Update the newly created booking

  • Guest booking will pop up and the following details needs to be updated:

    • Special Instructions - update Check in & Check Out Instructions. Refer to Special Instruction Guide (page 12) for the correct sequence.

    • Additional Items if any Add-Ons/requests were added and input/adjust fee under sales row if necessary

    • Update Rate under (3) Room Tab for private bookings

Step 6: Send booking confirmation to the email address provided.
Corporate Bookings

Step 1. Get all the important details from guest/booker:

  • Date of stay

  • Room type (bedding configuration)

  • How many pax

  • Check if guest is GC Member

  • Check if works for a company that has a contracted rate with us (only if they have not mentioned it yet)

Step 2. Check Availability & Rate

  • On HMS

    • Reservations → Rooms Availability → (3) Room Type Tab → check room type availability

    • Adjust date range if needed

    • Open Company Profile and check contracted rate. Check also if company has credit facility.

Step 3. Provide Rate

  • Always advise inclusions such as breakfast (depending on their contract), Wi-Fi, Newspaper, 1x 600ml water each day, complimentary welcome drink upon arrival (except government & Shipping companies), 10% off bill at Encore (except shipping companies).

  • We can match rates found online only if we are able to find it. If we match lower rate that guest has found online, ensure that corporate inclusions are also added, as this is part of their contract. When matching a lower rate for existing corporate companies, use the corporate rate code and adjust rate manually.

  • Corporate surcharges apply for special events.

Step 4. Create the booking

  • Follow same instruction in creating booking for private booking

  • Locate same company details you've pulled out when you checked rates

  • Rate Type

    • Choose the rate code that is in red coloured font

Step 5: Update the newly created booking

  • Guest booking will pop up and the following details needs to be updated:

    • Special Instructions - Check In, Check Out & Postings. Ensure that if there is chargeback for specific booking made to update booking with chargeback details and restrictions.

    • If guest is a regular guest, ensure to add VIP/Corp Guest on the Additional Items

    • Rate will automatically update with the contracted rate of company however this needs to be updated if surcharge applies.

Step 6: Send booking confirmation to the email address provided.
