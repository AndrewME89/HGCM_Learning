Creating a Reservation

Private Bookings
Step 1: Gather Guest Information
  • Date of stay

  • Room type (bedding configuration)

  • Number of guests (pax)

  • Check if the guest is a GC Member and request their membership number for confirmation.

Step 2: Check Availability & Rate
  • In iHMS:

    • Navigate to: Reservations → Rooms Availability → (3) Room Type Tab

    • Adjust the date range if necessary.

    • Check for rates and open the Rate Calculator (M: → Operations → Reservations → Promotions → Calculators → Promotional Calculator) to apply discounts.

      This should also be available on every Front Office Reception computer.

    • Ensure to apply an additional 10% off to the rates for GC Rewards members.

Step 3: Provide Rate
  • Always offer BAR & SAVER Rates and explain the difference between them.

  • We can match rates found online only if we are able to find them.

  • Corporate guests can utilize corporate rates for personal stays.

Step 4: Create the Booking
  • Press Alt + C or click the new reservation icon.

  • Fill in the necessary information:

    • **Guest:** Locate the guest if they have an existing profile.

      • Type the guest's name in the correct sequence (Last Name, First Name) and ensure it's spelled correctly.

      • Press F2 or the ellipses icon. A new window will pop up with a list of guest profiles. Choose the correct profile by matching contact number/company name. Edit the profile and add/update the credit card number to guarantee the booking.

      • If no existing profile is found, create a new one by clicking the new profile icon. Ensure all details are filled in:

        • VIP Status

        • Full Name (Last Name, First Name, Title)

        • Contact Number

        • Email (for booking confirmation purposes)

        • Credit card details

        • If the guest is a GC Member and has no existing profile, add their GC Membership # & ensure VIP status is VIP 4.

    • **Company Name:** Locate the guest’s company if it's already in the system.

      • Use 'Private Booking' as the company if the booking is made directly over the phone or through email.

      • Create the company if it's not yet in the system.

    • **Room Type:** The best rate type will be in red font. If multiple rates are available, select the best rate code that matches the quoted rate.

    • **Check In date**

    • **Check Out date**

    • **Number of Adults & Number of Children** (only if applicable)

    • **Rate type**

      • Click on the ellipses icon, and a list of rate codes will appear. Choose the best rate based on the quote given to the guest.

    • **Contact Person**

      • Enter the booker’s name and contact number under the contact person’s details.

      • If the guest staying is the same person making the booking, type 'self' under the contact person's details.

    • Once all details are filled in, press "Next" twice.

      • Mkt Seg 4 & Source will be filled in automatically.

    Step 5: Update the Newly Created Booking
    • The guest booking will appear. Update the following details:

      • **Communications:** Update Check In & Check Out Instructions. Refer to the Special Instruction Guide (page 12) for the correct sequence.

      • **Additional Items:** Add any Add-Ons/requests and input/adjust fees under the sales row if necessary.

      • **Update Rate:** Update the Rate under the (3) Room Tab for private bookings.

    Step 6: Send the booking confirmation to the email address provided.
    Corporate Bookings

    Step 1: Gather Guest Information
    • Date of stay

    • Room type (bedding configuration)

    • Number of guests (pax)

    • Check if the guest is a GC Member.

    • Check if the guest works for a company that has a contracted rate with us (only if they haven't mentioned it yet).

    Step 2: Check Availability & Rate
    • In iHMS:

      • Navigate to: Reservations → Rooms Availability → (3) Room Type Tab → check room type availability

      • Adjust the date range if needed.

      • Open the Company Profile and check the contracted rate. Also, check if the company has a credit facility.

    Step 3: Provide Rate
    • Always advise inclusions such as breakfast (depending on their contract), Wi-Fi, Newspaper, 1x 600ml water each day, complimentary welcome drink upon arrival (except for government & Shipping companies), and 10% off the bill at Encore (except for shipping companies).

    • We can match rates found online only if we can find them. If we match a lower rate that the guest has found online, ensure that corporate inclusions are also added, as this is part of their contract. When matching a lower rate for existing corporate companies, use the corporate rate code and adjust the rate manually.

    • Corporate surcharges apply for special events.

    Step 4: Create the Booking
    • Follow the same instructions for creating a booking for a private booking.

    • Locate the same company details you pulled out when you checked rates.

    • Rate Type:

      • Choose the rate code that is in red colored font.

    Step 5: Update the Newly Created Booking
    • The guest booking will appear. Update the following details:

      • **Communications:** Check In, Check Out & Postings. If there's a chargeback for a specific booking, update the booking with chargeback details and restrictions.

      • **Additional Items:** If the guest is a regular guest, add VIP/Corp Guest to the Additional Items.
